# eslint-config-node

Standard linting configuration for Node.js projects developed by STRG.

## Installation

```bash
# install with npm
npm install -D @strg/eslint-config-node

# or with yarn
yarn add -D @strg/eslint-config-node
```

## Usage

```js
// .eslintrc.js
module.exports = {
  extends: [
    '@strg/eslint-config-node'
  ]
}
```
